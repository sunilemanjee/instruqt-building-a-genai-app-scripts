Building Search AI (MCP) on Elasticsearch Serverless - Workshop Overview

Transform your search applications with AI-powered intelligence. This hands-on workshop teaches you how to build intelligent search systems using Elasticsearch Serverless, vector search, and AI models. Learn to deploy inference endpoints, implement semantic search, and create conversational interfaces that understand user intent. No prior AI experience required - just bring your curiosity and get ready to build the future of search.

Who Is This Workshop For?

Primary Audience:

  • Search Engineers building intelligent search applications
  • Solutions Architects designing AI-powered search systems
  • Backend Developers implementing semantic search capabilities
  • Data Engineers working with vector databases and embeddings
  • DevOps Engineers deploying serverless search infrastructure

Technical Background:

  • Basic Elasticsearch knowledge helpful but not required
  • Familiarity with APIs and JSON preferred
  • No prior AI/ML experience necessary
  • Suitable for intermediate to advanced technical professionals

What You'll Learn

Core Technologies

  • Elasticsearch Serverless - Configuration-free, auto-scaling search infrastructure
  • Vector Search - Semantic similarity using dense and sparse embeddings
  • AI Model Integration - ELSER, E5 multilingual, and custom ML models
  • Model Context Protocol (MCP) - Intelligent query processing layer
  • Semantic Reranking - AI-powered result optimization

Practical Skills

  • Deploy inference endpoints for multiple AI models
  • Build multilingual semantic search applications
  • Implement vector quantization for performance optimization
  • Create intelligent chatbots with grounded responses
  • Use ES|QL for modern query development

Advanced Features

  • Automatic text chunking and vectorization

  • Real-time query intelligence with MCP
  • Serverless auto-scaling and performance optimization
  • Production-ready search applications
