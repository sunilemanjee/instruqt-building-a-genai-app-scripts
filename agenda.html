Workshop Lab Agenda

Become a 5x Developer: Build Smarter Search AI on Serverless

A comprehensive hands-on journey through AI-powered search technologies, from basic setup to advanced machine learning integration.

10+ Labs
2-3 Hours
5+ AI Models
1
Demo

What are we building?

A short demo of the complete search application you will be creating throughout this workshop, showcasing the end-to-end AI-powered property search experience.

2
Infrastructure

Serverless

Explore Elasticsearch Serverless configuration-free approach where infrastructure decisions are automatically managed and abstracted away, eliminating operational overhead.

3
Data Exploration

View Properties Data

Use Kibana Discover to explore and understand the structure of the properties index data that will be used throughout the workshop exercises.

4
Search Basics

Playground - Find Homes

Use Elasticsearch Playground to perform basic property searches and understand traditional keyword-based search capabilities and limitations.

5
Foundation

AI & ML Models with Elastic

Understand why AI and ML are essential for modern search and how Elasticsearch's built-in capabilities dramatically improve search recall and precision.

6
Sparse Vectors

ELSER GPU Backed Inference Endpoint

Observe and test ELSER (Elastic Learned Sparse EncodeR) inference endpoint for semantic search using sparse vector representations and query expansion.

7
Dense Vectors

E5 Vector Inference Endpoint

Create and configure an E5 multilingual inference endpoint for dense vector embeddings, enabling cross-language semantic search capabilities.

8
LLM Integration

Completions Endpoint

Create and configure a Large Language Model (LLM) Chat Completion inference endpoint for conversational search and natural language response generation.

9
Reranking

Elastic Reranker Inference Endpoint

Create and configure an Elastic ReRanker inference endpoint to improve search result ordering using semantic similarity and relevance scoring.

10
Custom ML

ML Models in Search

Deploy and use custom machine learning models like NER, text classification, and zero-shot classification directly within Elasticsearch using Eland and ingest pipelines.

11
Semantic Fields

Properties Index - Semantic Text Fields

Explore semantic text fields in the properties index and understand how they enable automatic vectorization and semantic search capabilities.

12
Multi-Language

E5 Multi Language with ES|QL

Test the cross-language capability of E5 using ES|QL to perform semantic searches across different languages and demonstrate multilingual search effectiveness.

13
Performance

Quantization & Recall Impact

Explore the impact of vector quantization on search accuracy and recall in property search indices, balancing performance with search quality.

14
Relevance Engineering

Relevance Studio

Introduction to search relevance engineering using Elasticsearch Relevance Studio for systematic evaluation and improvement of search result quality.

15
Modern Querying

Search with ES|QL

Run semantic hotel searches with proximity-based geo filters using ES|QL, Elasticsearch's modern query language for complex search operations.

16
Business Logic

Query Rules

Create query rules that automatically promote sponsored properties based on search terms, implementing business logic directly into search results.

17
Search Enhancement

Synonyms UI

Learn to create and manage synonyms using Elasticsearch's user interface to improve search recall and handle vocabulary variations in property searches.

18
AI Integration

MCP & Intelligent Query Layer Chatbot

Build an intelligent chatbot using Model Context Protocol (MCP) and the intelligent query layer for conversational property search and natural language interactions.

19
Benchmark

Running a Vector Benchmark using ESRally

Optional lab to run vector search benchmarks using ESRally to measure and compare the performance of different vector search configurations and models.
